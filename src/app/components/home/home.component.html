Como Funciona o Sistema de Pontuação?

O Score do MMA Stats é calculado basicamente sobre os números de um lutador e, portanto, este é uma análise quantitativa e não qualitativa. É importante destacar também que, considerando que uma vitória dentro do UFC vale mais do que dentro de um evento regional, os números associados a uma luta são ponderados para melhor refletir esta realidade.
